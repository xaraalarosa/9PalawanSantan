Lesson 1.1: Quadratic Equations

Definition: A quadratic equation is any equation of the form ax2 + bx + c = 0, where a ≠ 0.

Standard Form: ax2 + bx + c = 0.

Let's Practice

1. Solve the quadratic equation: \(x^2 - 5x + 6 = 0\)

Answer: \(x = 2, 3\)

2. Determine if the following equation is quadratic: \(3x^2 + 2x - 1 = 0\)

Answer: Yes, it is a quadratic equation.

Lesson 1.2: Solving Quadratic Equations by Extracting the Square Roots

Method: Isolate the x2 term and take the square root of both sides.

Example: x2 = k ⇒ x = ±√k.

Let's Practice

1. Solve for \(x\): \(x^2 = 49\)

Answer: \(x = \pm 7\)

2. Solve for \(x\): \(4x^2 = 100\)

Answer: \(x = \pm 5\)

Lesson 1.3: Solving Quadratic Equations by Factoring

Method: Express the quadratic equation as a product of two binomials.

Example: ax2 + bx + c = 0 ⇒ (mx + n)(px + q) = 0.

Lesson 1.4: Solving Quadratic Equations by Completing the Square

Method: Transform the equation into a perfect square trinomial.

Steps:

  1. Move the constant term to the other side.
  2. Add and subtract (\frac{b}{2a})2 inside the equation.
  3. Factor the perfect square trinomial.
  4. Solve for x.

Lesson 1.5: The Quadratic Formula

Formula: x = \frac{-b ± √(b2 - 4ac)}{2a}.

Usage: Applicable for all quadratic equations.

Lesson 1.6: The Discriminant of the Quadratic Equation

Discriminant: D = b2 - 4ac.

Interpretation:

  • D > 0: Two distinct real roots.
  • D = 0: One real root (repeated).
  • D < 0: No real roots (complex roots).

Lesson 1.7: Sum and Product of the Roots of a Quadratic Equation

Sum: x1 + x2 = -\frac{b}{a}.

Product: x1 · x2 = \frac{c}{a}.

Lesson 1.8: Equations Transformable to Quadratic Equations

Method: Transform non-quadratic equations into quadratic form by substitution or algebraic manipulation.

Lesson 1.9: Quadratic Inequalities

Form: ax2 + bx + c > 0 or ax2 + bx + c < 0.

Solution: Find the roots of the corresponding quadratic equation and determine the intervals where the inequality holds true.

Lesson 2.1: Quadratic Functions

Definition: A quadratic function is a polynomial function of degree two, generally expressed as f(x) = ax^2 + bx + c, where a ≠ 0.

Standard Form: f(x) = ax^2 + bx + c

Vertex Form: f(x) = a(x - h)^2 + k

Factored Form: f(x) = a(x - r1)(x - r2)

Lesson 2.2: Graphs of Quadratic Functions

Parabola: The graph of a quadratic function is a parabola.

Vertex: The highest or lowest point on the graph, given by (h, k) in the vertex form.

Axis of Symmetry: The vertical line that passes through the vertex, x = h.

Direction: The parabola opens upwards if a > 0 and downwards if a < 0.

Lesson 2.3: The Vertex Form of Quadratic Functions

Vertex Form: f(x) = a(x - h)^2 + k

Vertex: (h, k)

Transformations:

  • Horizontal shifts: h
  • Vertical shifts: k
  • Reflections: If a is negative, the parabola reflects over the x-axis.
  • Stretching/Shrinking: The value of a determines the width of the parabola.

Lesson 2.4: The Vertex Form of Quadratic Functions (Continued)

Finding the Vertex: From the standard form f(x) = ax^2 + bx + c, the vertex can be found using h = -b / 2a and k = f(h).

Example: Convert f(x) = 2x^2 + 4x + 1 to vertex form.

Complete the square: f(x) = 2(x^2 + 2x) + 1

f(x) = 2((x + 1)^2 - 1) + 1

f(x) = 2(x + 1)^2 - 2 + 1

f(x) = 2(x + 1)^2 - 1

Vertex: (-1, -1)

Lesson 2.5: Equations of Quadratic Functions

Solving Quadratic Equations: Methods include factoring, using the quadratic formula, and completing the square.

Quadratic Formula: x = (-b ± √(b^2 - 4ac)) / 2a

Discriminant: Δ = b^2 - 4ac

  • If Δ > 0, two real solutions.
  • If Δ = 0, one real solution.
  • If Δ < 0, no real solutions (complex solutions).
